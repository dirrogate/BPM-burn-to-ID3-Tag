# BPM-burn-to-ID3-Tag
A vibe coded python utility to burn BPM to ID3 tags for MP3 because Rekordbox won't do it.

# Why this tool?
I use software for DJing in VR such as TribeXR and Vinyl Reality. It's PITA to have to go through Rekordbox and update its XML file, then upload it to the headset everytime a new song is "analyzed"
This tool will take a standard Rekordbox.xml file and once pointed to your main Music Folder, will iterate through and burn:
 1) The BMP found in the XML to the MP3 files ID3 tag
 2) Fill the Mp3's COMMENTS tag with BPM, Rating, Music Key
