# BPM-burn-to-ID3-Tag
A vibe coded python utility to burn BPM to ID3 tags for MP3 because Rekordbox won't do it.

# Why this tool?
I use software for DJing in VR such as TribeXR and Vinyl Reality. It's a PITA to have to go through Rekordbox and update its XML file, then upload it to the headset everytime a new song is "analyzed" just beacuse Rekordbox won't burn BPM to the MP3 file.

The tool could also be of use to other DJ software/hardware that does not rely on Rekordbox USB/XMLs.

This tool takes a standard Rekordbox.xml file and once pointed to your main Music Folder, will iterate through and burn:
 1) The BMP found in the XML to the MP3 files ID3 tag
 2) The Musical Key
 3) Fill the Mp3's COMMENTS tag with BPM, Rating, Music Key

**Vibe coded tool, so use at your own risk. Backup your main music folder just in case.**

This git hub repo serves as a storage for my own needs, while on the move. 🤘🎧

**Installation:**
Download the zip and extract. 
Run the exe file.
